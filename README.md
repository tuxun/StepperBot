Hardware needed:
2xUNL2003
1xArduino UNO (or anyother 5V atmega328p compatible board)
1x7805 Voltage regulator
1xbattery pack 7V to 12V
1xthermal heater sink
2x28BYJ48 5V 1:64 gear ratio64-5A (or 28BYJ64-5B)
